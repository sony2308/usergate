# User Registration Form
  This is a simple and responsive user registration form built using HTML, CSS(with Bootstrap) and JavaScript.
  The form includes fields for:
  - First Name & Last Name
  - Email Adress
  - Mobile Number
  - Password & Confirm Password
  - Gender selection
  - Address

## Features:
 - Client-side Validation for all required fields.
 - Error Messages Displayed below the inputs.
 - Password checks for strength and conformation.
 - Form resets after successful submission.
 - Styled with Bootstrap and custom Css for a clean UI

   This project is front-end only and does not require any server or database.
